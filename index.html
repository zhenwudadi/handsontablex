Documentation

handsontable(options)

Plugin constructor handsontable(options) accepts the following configuration options as the first argument:

  • rows - Initial number of rows
  • cols - Initial number of columns
  • minHeight - (Optional) If set, will add as many rows as needed to meet the given height in pixels
  • minSpareRows - (Optional) If set to 1 (or more), will add a new row at the end of grid when there are no more empty rows
  • onChange - (Optional) Function to be called after one or more fields is changed (with changes array as an argument)

handsontable("loadData", data)

Plugin method handsontable("loadData", data) accepts initial grid data as the argument.
