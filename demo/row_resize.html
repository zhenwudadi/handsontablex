Manual row resize

This page shows how to use Handsontable with row related built-in plugins:

  • Row resize

Row resize

To enable this experimental feature, use setting manualRowResize: true

In the bottom part of row header a draggable row resize handle will appear.

Double click on the row resize handle to automatically adjust row height.

State of the table has been restored.
