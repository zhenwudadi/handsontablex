Handsontable

a minimalistic Excel-like data grid editor for HTML, JavaScript & jQuery

Fixed rows and columns

Specify two fixed rows with fixedRowsTop: 2 and two fixed columns with fixedColumnsLeft: 2 option.

Note: You'll need horizontal scrollbars, so just set a container width and overflow: scroll in CSS.

For more examples, head back to the main page.
