Custom borders

To enable the custom borders feature, set the customBorders option. It could be set as true or an array of cells with borders arrays like: or

To initialize Handsontable with predefined custom borders, provide cells details in form of an array: { row: 2, col: 2, left:{ width:2, color: 'red' }, right:{ width:1, color: 'green' } } or range details in form of an array { range:{ from:{ row: 1, col: 1 }, to:{ row: 3, col: 4 } }, top:{ width: 2, color: '#5292F7' }, left: { width:2, color: 'orange' } }
