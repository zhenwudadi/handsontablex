Validators

Use the beforeValidate callback to manipulate value of changed cells before they are applied to the validator function.

NOTICE: this will not afect values of changes. This change value ONLY for validation!

Use the afterValidate callback to determinate if validation passed successfully or not.
You can cancel current change by returning false.
